How To Use
Paint Calc is designed to calculate the amount of paint required for painting walls. It will deduct any windows/doors measurements you input.
  1. Input Width and Height measurement for each wall.
  2. Input each window/door measurements.
  3. Click calculate.
All measurements should be in metres. Paint coverage information can be obtained from the manufacturer or the paint container. This is a estimate. Please confirm quantities with your supplier before purchasing.
